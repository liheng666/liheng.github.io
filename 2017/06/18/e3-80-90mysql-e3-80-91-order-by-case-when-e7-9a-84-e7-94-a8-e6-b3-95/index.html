【mysql】 order by case when 的用法

发表于 2017-06-18 | 分类于 mysql 
select * from table order by case field when field=2 then 1 else 0 end, field desc;

【解释】查询时，当某条数据中的字段field 的值为2时，该条field的排序权值为1，其他的为0，使用降序排列；
